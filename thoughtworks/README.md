[编译环境]:
    Linux
[语言选择]:
    C/C++
[编译]:
    make
[运行]:
    ./maze
[测试]:
    每个模块内部都有处理的输出结果,处理失败时返回false并退出
    打开Makefile文件中的DEF变量的值即可看到每个模块的输出
[程序输入]:
    maze.input文件中的内容即为文件输入
[程序输出]:
    输出到标准输入
[程序模块]:每个模块对应一个.cpp文件,模块接口与文件名一致
    [模块号]:[对应函数]
        [描述]
    [1]:[GetInput()]
        从文件获取输入
    [2]:[isFormatLeagal()]
        判断输入格式是否符合特定规则
    [3]:[isValueLeagal()]
        判断输入的值是否合法
    [4]:[canRoadConnect]
        判断输入的各个通路是否符合连通规则
    [5]:[CreateMaze()]
        创建迷宫图
    [6]:[ConnectMaze()]
        按照输入的通路将迷宫连接
    [7]:[PrintMaze()]
        打印迷宫图
    [8]:[maze_perror()]
        根据错误码打印错误信息

[程序流程]:
    1.从输入文件获取输入,失败会退出并打印错误信息
    2.判断输入格式是否符合规则,不符合程序退出并打印错误信息
    3.判断输入的值是否在超出范围,超出程序退出并打印错误信息
    4.判断每条通路是否可以连通,不可以连通程序退出并打印错误信息
    5.创建迷宫图
    6.连接迷宫所有通路
    7.打印迷宫图
    8.程序结束,释放内存并退出
